# **Énoncé**

Étant donné un nombre à 3 chiffres, affichez "OUI" si ses trois chiffres sont dans l'ordre strictement croissant de gauche à droite ou "NON" sinon.

# Exemple d'entrée #1

```
179
```

# Exemple de sortie #1

```
OUI
```

# Exemple d'entrée #2

```
197
```

# Exemple de sortie #2

```
NON
```
