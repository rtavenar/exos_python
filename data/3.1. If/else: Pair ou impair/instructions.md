# Consigne

Étant donné un entier, afficher "pair" s'il est pair ou "impair" sinon.

# Exemple d'entrée #1

```
5
```

# Exemple de sortie #1

```
impair
```

# Exemple d'entrée #2

```
6
```

# Exemple de sortie #2

```
pair
```
