# **Énoncé**

Étant donné un nombre à 4 chiffres, affichez "OUI" si c'est un palindrome (c'est-à-dire qu'on obtient la même valeur en le lisant de droite à gauche ou de gauche à droite) et "NON" sinon. 

# Exemple d'entrée #1

```
1221
```

# Exemple de sortie #1

```
OUI
```

# Exemple d'entrée #2

```
1234
```

# Exemple de sortie #2

```
NON
```
