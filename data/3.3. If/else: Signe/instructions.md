# Énoncé

Étant donné un entier, afficher `1` s'il est positif, `-1` s'il est négatif, ou `0` s'il est nul.

Essayez d'utiliser la syntaxe `if-elif-else` pour cela.

# Exemple d'entrée

```
179
```

# Exemple de sortie

```
1
```
