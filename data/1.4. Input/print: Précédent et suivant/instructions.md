# Consigne

Écrire un programme qui lit un nombre entier et affiche le nombre précédent et le nombre suivant (Cf. exemple ci-dessous).

# Exemple d'entrée

```
179
```

# Exemple de sortie

```
Le nombre suivant 179 est 180
Le nombre précédant 179 est 178
```
