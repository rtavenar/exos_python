# Consigne

À partir d'une liste de nombres entiers donnés en entrée (chaque nombre est séparé par un espace), trouver et afficher tous les éléments qui sont plus grand que leur voisin de gauche.

# Exemple d'entrée

```
1 5 2 4 3
```

# Exemple de sortie

```
5 4
```
