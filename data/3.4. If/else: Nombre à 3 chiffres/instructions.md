# Énoncé

Étant donné un entier, affichez "OUI" s'il est composé de 3 chiffres et "NON" sinon.

# Exemple d'entrée #1

```
179
```

# Exemple de sortie #1

```
OUI
```

# Exemple d'entrée #2

```
1234
```

# Exemple de sortie #2

```
NON
```
