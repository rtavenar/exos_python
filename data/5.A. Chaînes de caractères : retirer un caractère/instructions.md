# Consignes

Étant donnée une chaîne de caractères entrée au clavier, retirer tous les caractères `@` de cette chaîne.

# Exemple d'entrée

```
Bilbo.Baggins@bagend.hobbiton.shire.me
```

# Exemple de sortie

```
Bilbo.Bagginsbagend.hobbiton.shire.me
```
