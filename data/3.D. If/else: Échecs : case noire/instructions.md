# **Énoncé**

Étant donnés deux entiers indiquant une case de l'échiquier, affichez OUI si c'est une case noire (marron sur le dessin ci-dessous) ou NON sinon.

 

# **Exemple d'entrée** #1

```
1
1
```

# **Exemple de sortie** #1

```
OUI
```

# **Exemple d'entrée** #2

```
1
2
```

# **Exemple de sortie** #2

```
NON
```
