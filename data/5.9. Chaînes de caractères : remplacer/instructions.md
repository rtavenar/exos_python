# Consignes

Étant donnée une chaîne de caractères entrée au clavier contenant au moins une fois le caractère `1`, remplacer dans cette chaîne tous les nombres `1` par le mot `one`.

# Exemple d'entré

```
1+1=2
```

# Exemple de sortie

```
one+one=2
```
