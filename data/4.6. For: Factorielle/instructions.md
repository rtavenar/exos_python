# Consigne

En mathématique, la factorielle d'un entier n (n!) est le produit suivant :

n! = 1 × 2 × … × n

Pour l'entier n donné en entrée, calculer la valeur de **n!**. Ne pas utiliser le module math dans cet exercice.

# Exemple d'entrée

```
4
```

# Exemple de sortie

```
24
```
