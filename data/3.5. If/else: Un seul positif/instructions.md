# **Énoncé**

Étant donnés deux entiers non nuls, affichez "OUI" si l'un des deux est positif (mais pas les deux) et "NON" sinon.

# Exemple d'entrée #1

```
-5
```

```
10
```

# Exemple de sortie #1

```
OUI
```

# Exemple d'entrée #2

```
5
```

```
10
```

# Exemple de sortie #2

```
NON
```
