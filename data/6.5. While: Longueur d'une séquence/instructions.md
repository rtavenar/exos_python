# **Énoncé**

Étant donnée une séquence d'entiers positifs, où chaque entier est écrit sur une ligne distincte, affichez le nombre de valeurs lues **avant** de rencontrer un 0.

# **Exemple d'entrée**

```
1
7
9
0
5
```

# **Exemple de sortie**

```
3
```
