# **Énoncé**

Étant donnée une séquence d'entiers positifs, où chaque entier est écrit sur une ligne distincte, affichez le nombre de valeurs paires lues **avant** de rencontrer un 0.

# **Exemple d'entrée**

```
2
1
4
0
```

# **Exemple de sortie**

```
2
```
