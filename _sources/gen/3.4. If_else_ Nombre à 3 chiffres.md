# 3.4. If/else: Nombre à 3 chiffres

## Énoncé

Étant donné un entier, affichez "OUI" s'il est composé de 3 chiffres et "NON" sinon.

## Exemple d'entrée #1

```
179
```

## Exemple de sortie #1

```
OUI
```

## Exemple d'entrée #2

```
1234
```

## Exemple de sortie #2

```
NON
```

````{dropdown} Proposition de solution

```python
a = int(input())
if a >= 100 and a < 1000:
  print('OUI')
else:
  print('NON')
```
````
