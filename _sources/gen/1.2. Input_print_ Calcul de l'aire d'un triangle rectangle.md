# 1.2. Input/print: Calcul de l'aire d'un triangle rectangle

## Consigne

Écrire un programme qui lit la longueur de la base et la hauteur d'un triangle rectangle et affiche l'aire. Chaque nombre sera entré séparément.



## Exemple d'entrée

```
3
5
```

## Exemple de sortie

```
7.5
```
