# 3.8. If/else: Minimum de 3 nombres

## **Énoncé**

Étant donnés trois entiers, affichez le plus petit des 3.

## Exemple d'entrée

```
5
3
7
```

## Exemple de sortie

```
3
```

## Aide

http://rtavenar.github.io/teaching/python_poly/html/poly.html#structures-de-contrôle

## Squelette

```python
# Lire un entier :
# a = int(input())
# Afficher une valeur :
# print(a)
```

````{dropdown} Proposition de solution

```python
a = int(input())
b = int(input())
c = int(input())

if a < b and a < c:
  print(a)
elif b < a and b < c:
  print(b)
else:
  print(c)
```
````
