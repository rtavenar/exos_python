# 2.8. Nombres : siècle

## Consigne

Étant donnée une année (entrée au clavier comme un entier positif) trouver et afficher le siècle correspondant. Noter que, par exemple, le 20ème siècle commence avec l'année 1901.

## Example input

```
2000
```

## Example output

```
20
```
