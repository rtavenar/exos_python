# 6.3. While: Puissances de 2

## **Énoncé**

Étant donné un entier X, on cherche le plus grand entier **n** pour lequel **2ⁿ** est inférieur ou égal à X. Affichez **n** et **2ⁿ**.


## **Exemple d'entrée**

```
50
```

## **Exemple de sortie**

```
5 32
```

````{dropdown} Proposition de solution

```python
x = int(input())
n = 1
while 2 ** n <= x:
  n += 1
print(n - 1, 2 ** (n - 1))
```
````
