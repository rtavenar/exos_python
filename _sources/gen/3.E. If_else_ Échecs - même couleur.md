# 3.E. If/else: Échecs - même couleur

## **Énoncé**

Étant données deux cases d'un échiquier, afficher `OUI` si elles sont de même couleur et `NON` sinon.



## **Exemple d'entrée**

```
1
1
2
6
```

## **Exemple de sortie**

```
OUI
```

````{dropdown} Proposition de solution

```python
x1 = int(input())
y1 = int(input())
x2 = int(input())
y2 = int(input())

if (x1 + y1) % 2 == (x2 + y2) % 2:
  print('OUI')
else:
  print('NON')
```
````
